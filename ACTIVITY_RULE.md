# Activity Rule

## 넘버스 수업의 목적
 1. 학생들의 학습 활동의 누적
 2. 불필요한 종이 낭비 방지
 3. 아이패드와 애플펜슬을 활용한 쾌적한 학습 환경 제공
 4. 무한한 넘버스 캔버스를 바탕으로 학생의 다양한 창의적 활동 지원
 5. 학생의 사고 과정을 기록하고 평가
 6. 학생과 교사 간 공유 파일 체계를 통한 피드백 제공 및 수용
 7. 학습에 필요한 다양한 영상, 음원, 사이트 자료 제공
 8. 학습자 주도적인 환경 구축 및 개별화된 자료 사용 및 개별화된 수업 진행.

## 넘버스 활동의 구성(객체)
1. 교과서
    - 실제 교과서 pdf에서 텍스트를 추출하여 넘버스 시트에 삽입
    - 차시별로 구분, 한 차시의 교과서는 한 시트에 삽입
    - 전 과목 공통
2. 추가자료
    - 사회과 부도, 수학익힘책, 실험관찰 등 각 과목에 2권 이상의 교과서가 있는 경우 해당 차시에 맞는 교과서 추가 삽입
    - 교과서 하단에 추가자료 교과서 배치
3. 학습 노트
    - 줄 없는 백지 노트로 구성. 줄이 없어야 애플펜슬을 활용한 필기에 자유도를 줄 수 있음.
    - 학습 노트에는 교사의 설명 내용을 적거나, 학생이 자유롭게 필기할 수 있도록 함.
    - 노트 하단에 메모, 질문 영역을 추가
    - 노트 상단에 차시, 학습 목표를 표시
    - 노트 우측에는 질문의 유형을 배치하여, 학생들이 질문을 쉽게 만들 수 있도록 제시
4. 활동 영역
    - 교과서 활동 이외의 추가 활동을 제시하는 공간.
    - 주로 넘버스 활용, 키노트 활용, 페이지스 활용, 개러지 밴드 활용 등 교과 내용에 추가 보완 및 심화 성격을 지니는 다양한 활동을 제시
    - 활동의 난이도는 학생들의 수준에 맞게 조절함.
    - 활동은 크게 3가지 종류로 구성할 수 있음. 
        1. 학생의 창의력을 요구하는 창작 활동 -> 그림, 글, 영상 등으로 배운 내용을 표현하는 하이터치 활동
        2. 학생의 사고력과 문제 해결 능력을 요구하는 활동 -> 깊이 사고하여 기존 개념들과 지식 간의 연결을 통해 해결하는 활동, 글쓰기 활동
        3. 기초기능의 연습을 위한 활동 -> 문제 풀이, 단어 암기, 자료 조사 등 교육과정에 제시된 과목별 기초 기능을 연습할 수 있는 활동
    - 학생의 창의력을 요구하는 창작 활동을 제시할 때는 충분한 가이드라인과 예시 자료를 함께 제공
    - 학생의 사고력과 문제 해결 능력을 요구하는 활동을 제시할 때는 충분한 사고 과정과 문제 해결 과정을 학생이 직접 애플펜슬로 기록할 수 있도록 체계적인 구성
    - 기초기능의 연습을 위한 활동에서는 충분한 연습 공간을 제공
    - 가능하면 학생들이 수준별로 자신이 원하는 활동을 통해 지식을 구성하고 표현할 수 있게 다양한 활동을 제시
5. 참고 자료 영역
    - 활동에 필요한 참고 자료를 제시하는 공간.
    - 참고 자료의 유형은 아래와 같은 것들이 있음.
        1. 글
        2. 그림 및 사진
        3. 유튜브 영상
        4. 웹사이트 링크
        5. 역사적 사료
    - 이 이외에도 교과 학습에 필요한 경우 다양한 형태의 자료를 제시할 수 있음.
    - 참고 자료는 학생들이 활동에 쉽게 접근할 수 있도록 배치
    - 참고 자료의 목적은, 학생들이 모든 자료를 보는 것이 아니라, 필요할 때 적절히 활용할 수 있도록 하는 것임.
6. 학습지 영역
    - 학습지 영역은 활동 영역과 별도로 구성
    - 학습지 영역은 학생들이 활동을 통해 얻은 지식을 바탕으로 문제를 풀거나, 활동을 통해 얻은 지식을 바탕으로 글을 쓰는 등 다양한 활동을 할 수 있도록 구성
    - 학습지 영역에는 각 교과서 사이트에서 공식적으로 제공하는 학습지를 사진으로 넣거나, 넘버스로 직접 제작하여 넣음.
    - 학습지 영역의 구성은 좌측에 학생이 해결해야 하는 학습지, 우측에 학습지의 답안지를 배치함.
7. AI 코스웨어 연계 영역
    - AI 코스웨어 연계 영역은 활동 영역과 별도로 구성
    - 학급에서 각 과목에서 사용하는 AI코스웨어 사이트로 향하는 링크 버튼을 제공
    - 링크 버튼 하단에는 코스웨어 풀이 및 채점 결과를 기록할 수 있는 영역

## 넘버스 제작 시 객체의 배치
1. 수업 전 - 수업 중 - 수업 후 정리 순으로 가로로 배치
2. 각 객체는 200pt 간격으로 배치
3. 수업 전 객체에는 교과서, 추가자료를 배치
4. 수업 중 객체에는 학습노트, 활동영역을 배치
5. 수업 후 정리 객체에는 참고자료, 학습지, AI 코스웨어 연계 영역을 배치

## 넘버스 제작 시 객체의 크기
- 각 객체의 크기는 내용에 따라 유기적으로 조절하되, 기본값은 가로 1000pt, 세로 700pt로 설정한다.
